v.js
====

Increase / set [version numbers](http://semver.org/) in package.json.


### Install ###

```bash
$ npm -g install v.js

```

### Use ###

```bash
v.js -h   # Help
v.js -M   # Increment Major version
v.js -m   # Increment Minor version
v.js -p   # Increment Patch number
v.js --set 1.2.3 # Manually set version

```
